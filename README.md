# CODEVERSE

**Author:** Sunidhi Sourabh  
**Goal:** A curated collection of programming problems, from foundational C programs to real contest/DSA problems, demonstrating problem-solving and coding skills.

---

## 🌟 Overview

CODEVERSE is a structured repository to showcase programming expertise. It is divided into sections:

1. **Curriculum:** Core C programming exercises to build a strong foundation.  
2. **Platform Problems:** Solutions from coding platforms like Unstop, LeetCode, CodeChef, and Codeforces.  
3. **Concepts Demonstrated:** Loops, conditionals, arrays, strings, frequency counting, and control structures.  

This repository is designed for **learning, portfolio building, and preparing for valued internships.**

---

## 🧠 Key Learning Outcomes

- **Control Flow:** if–else, switch, loops  
- **Arrays & Strings:** traversal, frequency counting, string length  
- **Mathematics:** sum of natural numbers, area calculations, grading logic  
- **Memory Concepts:** sizeof operator, data type sizes  
- **Problem Solving Skills:** Mapping real-world problems to code, handling input/output efficiently
